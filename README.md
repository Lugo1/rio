# Proyecto RIOTEC

Este es un proyecto Fullstack desarrollado con:

- **Frontend**: React.js + Vite
- **Backend**: Node.js + Express + PostgreSQL
- **Base de datos**: Railway
- **Despliegue**:
  - Frontend en Vercel
  - Backend en Render
  - PostgreSQL en Railway
